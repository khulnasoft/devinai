### `@https://your-link.com`

In order for Devinai to visit a link before responding, type `@` and then paste in the link.

<Frame>
    <img src="/images/context/@link.png" />
</Frame>

### Remove Links

By default, we automatically parse links and turn them into `@Links` in Devinai Chat.
If you prefer to have the link as plain text, click on the link and then click `Unlink`.
