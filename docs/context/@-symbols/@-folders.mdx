<Info>Currently, `@Folders` is only supported in the Devinai Chat.</Info>

### `@Folders`

You can also reference entire folders in Devinai as context. `@Folders` is especially
useful for [long context chat](/chat/overview#long-context-chat) where you want to provide a lot of context to the AI.
