---
title: 'Rules for AI'
---

You can add custom instructions to Devinai by modifying the `Rules for AI` section under `Devinai Settings` > `General` > `Rules for AI`.

This custom instruction will be included for features such as Devinai Chat and Ctrl/⌘ K.

<Frame>
    <img src="/images/context/rules-for-ai.png" />
</Frame>

### `.devinairules`

For project-specific instructions, you can include the instructions in a `.devinairules` file in the root of your project.

As the same as the "Rules for AI" section, the instructions in the `.devinairules` file will be included for features such as Devinai Chat and Ctrl/⌘ K.
